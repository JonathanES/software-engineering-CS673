insertNewProject

insertNewProject(userID: any, projectName: any, dueDate: any)
Parameters
userID (any) of the user that wants to connect to the application
projectName (any) name of the project to be added
dueDate (any) of the project we are adding 1) user enters the name and dueDate for the project 2) query assigns the time of the submission as the DateCreated for the project 3) as default the project's IsDeleted mark as false 4) after the first query which adds the project to Projects table second query add the user who created the project as admin in ProjectUsers table 5) function returns the result of the first query as an output 6) this function can be called from Frontend as follows

client.on('USER_CREATE_PROJECT', async (userID, projectName, dueDate ) => { const result = await projectController.insertNewProject(userID,projectName,dueDate); client.emit('CREATE_PROJECT', result); })

insertNewTask

insertNewTask(parentID: any, categoryID: any, userID: any, statusID: any, priorityID: any, taskName: any, taskInfo: any, expectedDuration: any, actualTimeSpent: any)
Parameters
parentID (any) of the task, if a task is a subtask of a task this should point to parent Task
categoryID (any) categoryID of the task that is belong to
userID (any) of the user who this task is assigned to
statusID (any) of the task, task status can be 1 ->'did not start, 2-> working on it, 3 -> done
priorityID (any) of the task, priority of the task can be 1-> low, 2-> Moderate, 3-> High
taskName (any) Name of the task, not null
taskInfo (any) Information about the task, not null
expectedDuration (any) expected time for the given task
actualTimeSpent (any) actual time spent on the task 1) user enters the above parameters as input 2) query adds the task under the selected category of the selected project 3) this function can be called from Frontend as follows

client.on('USER_CREATE_TASK', async (parentID, categoryID, userID, statusID, priorityID, taskName, taskInfo, expectedDuration, actualTimeSpent) => { const result = await taskController.insertNewTask(parentID, categoryID, userID, statusID, priorityID, taskName, taskInfo, expectedDuration, actualTimeSpent); client.emit('CREATE_TASK', result); })
